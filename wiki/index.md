# Risa's All In One Menu

**One launcher hotkey (F1) opens many mod menus — and frees each mod's own hotkey for other use.**

Tired of remembering a different hotkey for every mod menu, or fighting keybind conflicts? Risa's All In One Menu brings supported SKSE mod menus together under one unified launcher, keeping your keyboard free for gameplay, other mods, and custom bindings.

<div class="grid cards" markdown>

- :material-keyboard: **One key for everything**

    Press **F1** to open the launcher, pick the menu you want. Press **F1** again or **Escape** to close it.

- :material-key-variant: **Frees your keys**

    Each supported menu's original hotkey is blocked or relocated so you can rebind it to anything else.

- :material-api: **Uses official APIs**

    Where a mod exposes an API, the launcher uses it for reliable opening, closing, and state tracking.

- :material-backup-restore: **Clean uninstall**

    A one-click restore returns every managed mod to its original hotkey before you remove the mod.

</div>

## Quick start

1. Install with a FOMOD-aware mod manager (see [Installation](installation.md)).
2. Launch Skyrim, reach the main menu, then exit and relaunch once so newly managed menus settle.
3. In game, press **F1** to open the launcher.
4. Click a mod's button to open its menu. Press **F1** again or **Escape** to close.

!!! tip "The launcher hotkey is configurable"
    F1 is the default, but you can change it — and add Ctrl / Shift / Alt, double-tap, and Easy Close options — from the launcher's **Settings** tab.

## Key features

- Launch supported mod menus from one interface.
- Dedicated sub-menus for **Community Shaders** and **Skyrim Party Sheet**.
- Rebind supported menu hotkeys directly from the **Settings** tab.
- Blocks or relocates supported menu hotkeys to reduce conflicts and free those keys.
- Uses official menu APIs where available; other integrations use direct menu control or a single targeted config edit.
- Custom hotkeys are saved and carry over between updates and new games.
